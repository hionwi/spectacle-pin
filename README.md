# spectacle-pin
1. mkdir ~/Scripts
2. cd ~/Scripts && git clone https://github.com/hionwi/spectacle-pin.git .
3. 添加 k-copy.sh （复制截图） 和 k-pin.sh （贴住截图） 的快捷键，支持滚轮缩放，Esc或者鼠标右键退出 / Add keyboard shortcuts for k-copy.sh (Copy Screenshot) and k-pin.sh (Pin Screenshot). Support mouse wheel zooming, and exit with the Esc key or right-click.
